## 구현 기능 목록
### 서비스
- [X] 1부터 45까지 번호를 뽑는다.
- [X] 숫자가 6개인 로또 번호를 뽑는다. 
- [X] 입력 받은 만큼 로또 번호를 뽑는다.
- [X] 입력 받은 금액에 맞는 구매 개수만큼 로또를 반환한다.
- [ ] 당첨 번호랑 보너스 번호를 저장한다.
- [ ] 당첨 번호랑 로또 번호를 비교하고 결과를 반환한다.
- [ ] 총 수익률을 계산한다.
### 입력
- [ ] 금액을 입력받는다.
- [ ] 당첨 번호를 입력받는다.
- [ ] 보너스 번호를 입력 받는다.
### 출력
- [ ] 구매한 로또들을 출력한다.
  - [ ] 구매한 장수를 출력한다.
  - [ ] 구매한 번호들을 출력한다.
- [ ] 당첨 통계를 출력한다.
  - [ ] 당첨 개수를 출력한다.
  - [ ] 수익률을 출력한다.
